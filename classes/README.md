# vehiculo

Genera un programa con esta estructura.

Genera un método  que permita visualizar los datos más importantes del vehículo, de diferente tipo de acuerdo a la clase.

En el programa principal genera una instancia para cada clase e ingresa los datos del teclado, envialos a los atributos pero no con el constructor, utilizando la instancia generada.

En el main solo deben verse llamadas a los métodos.
